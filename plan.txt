# 3️⃣ **Agent System (MVP Version)**

### Only 5 core agents are required. Lightweight, simple, and modular.

---

## **1. Storyboard Agent (CPU)**

**Purpose:** Convert user prompt → structured plan.

**Inputs:**

* Prompt
* Style
* Duration

**Outputs:**

```
{
  "scenes": [
    { "id": 1,
      "description": "Person jogging in park...",
      "key_visual": "dynamic morning workout scene",
      "script": "Meet FitFlow — your personal fitness app…" }
  ]
}
```

**Model Choice:**

* GPT-4o-mini (best balance)
* OR Llama 3.1 70B (free on Groq)

**Runs on:** CPU worker (no GPU needed)

---

## **2. Image Generation Agent (GPU)**

**Purpose:** Convert scene descriptions → images.

**Model Options:**

* **FLUX 1-dev** (best open-source quality, fast)
* SD3 (if GPU weaker)

**Outputs:**
1 high-quality keyframe per scene.

**Runs on:** GPU worker

---

## **3. Animation Agent (GPU)**

**Purpose:** Convert keyframe → 3–6 sec animation.

**Model Options:**
✔ **SVD (Stable Video Diffusion)** — free, easy, lighter
✔ **LTXVideo** — fast, great motion
✔ **Open-Sora 1.2** — if available

**Why:**
Fast, clean, 5–7 sec animation good enough for MVP.

**Runs on:** GPU worker
(Can use same GPU as Image Agent or separate queue.)

---

## **4. Audio Agent (CPU)**

### Sub-agents:

#### **A. Voice Agent**

Model: XTTS or Piper
Input: Scene script
Output: narration.wav

#### **B. Music Agent**

Model: MusicGen
Input: “upbeat cinematic background music”
Output: bg_music.wav

Both run on CPU, no GPU needed.

---

## **5. Editor Agent (CPU)**

**Purpose:** Combine animation + voice + music into final MP4.

Using **FFmpeg (NVENC for GPU acceleration where possible)**

Steps:

1. Combine all animation clips
2. Lay voice track
3. Add background music
4. Crossfade transitions
5. Export final.mp4

**Runs on:** CPU worker
(GPU optional for NVENC but not required)

# 4️⃣ **Full End-to-End MVP Workflow**

Below is the execution chain from click → final video:

```
User → Streamlit
        ↓
FastAPI receives prompt
        ↓
Redis job created (job_id)
        ↓
Celery workers pick tasks:

1. Storyboard Agent
   ↓ produces scenes.json

2. Image Agent  
   ↓ produces scene1.png, scene2.png…

3. Animation Agent  
   ↓ produces scene1.mp4, scene2.mp4…

4. Audio Agent  
   ↓ produces voiceover.wav + music.wav

5. Editor Agent  
   ↓ final_video.mp4

Storage → R2
Streamlit UI fetches + displays result
```
